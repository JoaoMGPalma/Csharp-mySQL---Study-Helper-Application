# Csharp-mySQL---Study-Helper-Application
A robust desktop quiz application developed in C# using Windows Forms and MySQL. This project was designed to facilitate study through interactive quizzes, featuring full database integration for managing users, questions, and results.

Key Features
- User Authentication: Secure login and registration system for students and administrators.
- Dynamic Quiz Engine: Questions are loaded from a MySQL database, supporting multiple themes and randomized answer selection.
- Administrative Panel: A dedicated management interface that allows administrators to add/edit users, modify quiz questions, and monitor performance metrics.
- Result Tracking: Automatically calculates final scores and stores historical results for user progress analysis.
- Custom UI/UX: Features a polished visual interface with custom icons and elements designed with support from Adobe Express.

Technical Stack
- Frontend: C# Windows Forms (.NET).
- Backend Database: MySQL.
- Database Connectivity: Utilizes the MySql.Data library for seamless SQL integration and CRUD operations.
- Architecture: Implements custom classes like Resposta (Answer) to handle complex data logic and boolean state management.

Prerequisites
- Visual Studio (2022 recommended).
- MySQL Server.
- MySQL Connector/NET (MySql.Data NuGet package).
